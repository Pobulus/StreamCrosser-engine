Stream Crosser

Stream Simulator wyświetla listę zdefiniowanych strumieni wejściowych i umożliwia przesłanie pojedynczego wiersza danych lub ustawienie automatycznego wysyłania co zadany czas
Query Panel pozwala sparsować zapytanie i zainstalować je w silniku, wyświetla podgląd grafu
Output Panel wyświetla wyniki wyjść zdefiniowanych w obecnym zapytaniu
